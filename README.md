# Engineering Test UI

This is my UI for the engineering test found [here](https://github.com/ECliamb/engineering-test).



## Setup

Add a .env file with the following:

```env
NEXT_PUBLIC_API_URL=http://localhost:3001
```

To install dependencies:

```bash
npm install
```

To build the app:

```bash
npm run build
```

To start the app:

```bash
npm start
```
